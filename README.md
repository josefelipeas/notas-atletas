# Pontuação dos Atletas

Aplicação que apresenta as notas dos atletas de uma competição de ginástica rítmica.

## Funcionalidade

A funcão recebe uma matriz de objetos contendo o nome e as notas desses atletas. Manipula as notas para que sejam computadas apenas as notas válidas, desconsiderando a menor e maior nota. Efetua o cálculo das médias, e faz uma breve apresentação ao usuário utilizando o console.log(), mostrando respectivos nomes, notas e médias.

## Como executar

1. Clone o repositorio `https://github.com/josefelipeas/notas-atletas.git`
2. Instale o ambiente de execução [node](https://nodejs.org/pt/download).
3. Execute o seguinte comando no terminal, no diretório correspondente ao projeto:
```bash
node notas-atletas.js
```
